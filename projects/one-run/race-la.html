ABOUT TFC GLOBAL KAPAMILYA RUN

The TFC Global Kapamilya Run is organized by The Filipino Channel and ABS-CBN Foundation International. This run is part of “Mga Kuwento ng Kasiyahan: The Grand Kapamilya Weekend,” an event celebrating ABS-CBN’s 60th anniversary on television. They are also in solidarity with other running events in Quezon City, Bacolod, Cebu and Davao collectively dubbed “One Run, One Philippines.”

Proceeds from the TFC Global Kapamilya Run in Los Angeles, California will be used in environmental projects adopted by ABS-CBN Lingkod Kapamilya Foundation as part of Bantay Kalikasan and RNG’s thrust to promote eco-tourism.
